My dotfiles
===========

Setup
-----

case github.com
~~~~~~~~~~~~~~~

.. code-block:: bash

    git clone https://github.com/kashewnuts/dotfiles ~/dotfiles
    ~/dotfiles/setup.sh

case bitbuket.org
~~~~~~~~~~~~~~~~~

.. code-block:: bash

    hg clone https://bitbucket.org/kashewnuts/dotfiles
    ~/dotfiles/setup.sh
