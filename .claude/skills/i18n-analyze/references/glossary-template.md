# プロジェクト用語集テンプレート

翻訳の一貫性を保つための用語集。プロジェクト固有の用語をここに定義する。

## 使い方

1. このテンプレートをプロジェクトにコピー
2. プロジェクト固有の用語を追加
3. 翻訳時に参照

## 用語集フォーマット

```markdown
| 日本語 | 英語 | 備考 |
|--------|------|------|
| イベント | event | 固有名詞として扱う |
| グループ | group | 組織を指す場合 |
| 参加 | join / participate | 文脈による |
```

## 翻訳ルール

1. **一貫性**: 同じ用語は常に同じ翻訳を使用
2. **文脈**: 文脈によって訳し分けが必要な場合は備考に記載
3. **固有名詞**: サービス名・ブランド名はそのまま使用
4. **敬語**: 英語では丁寧な表現に統一（please, would you like to など）
5. **変数**: `%(name)s` などの変数はそのまま維持
